
# Trabalho da Prova 2 de POO

lembrando dos requisitos: Requisitos técnicos
O projeto deve ser desenvolvido em PHP.
O uso do Composer é obrigatório para organização e/ou dependências.
O sistema deve aplicar os quatro princípios POO: abstração, herança, encapsulamento e polimorfismo.
O sistema deve ser uma aplicação CLI (Command-Line Interface), portanto será executada diretamente pelo terminal.
Faça uso da função readline para fazer o input do teclado.
Faça uso de type castings para converter a entrada proveniente do teclado.
O sistema deve ter opções para cadastrar, editar e excluir objetos.
Exemplo: se a temática do sistema é relacionado a esportes, então precisa ter o cadastro de jogadores, equipes, técnicos etc.
O uso de arrays é obrigatório.
Mínimo de 7 classes ao todo.
